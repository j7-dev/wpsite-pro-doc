# 創建新課程

在PowerCourse中，可創建無數的**主題課程**，

在每個主題課程下，至少會有１個**章節**，章節下可創建無數個**單元**。



而你的課程影片都會放在**單元**中，如有任何課程內容要做編輯，

都請先找到該**主題課程**，點一下課程標題就可開始編輯囉！

接下來就帶大家走一遍創建新課程與單元的流程



{% hint style="info" %}
為了在創建新課程時能更順暢，可先到以下這些地方先創立新課程中會用到的欄位資料：

**・課程分類/標籤** - 建立課程分類/標籤，例如付費課程、免費課程、架站課程等

**・講師管理** - 建立講師資訊

{% endhint %}



## <mark style="color:orange;">怎麼建立新課程？</mark>

1. 進入課程後台
2. 點擊 課程列表 / 新增課程
3. 列表會跳出 新課程，點標題繼續編輯

**銷售網址** - 這裡你可以自定義課程頁面後面的網址名稱，想取什麼都可以

小提醒：銷售網址欄位最右邊的<img src="../../.gitbook/assets/copy.png" alt="" data-size="line">按鈕可複製整段網址

### <mark style="color:blue;">課程描述</mark>

這裡可以編輯 課程頁面 的所有資訊



1. **課程名稱** - 放課程正式名稱<br>
2. **商品分類 / 標籤**  - 可選擇課程是屬於哪個分類或標籤\

   ❯❯ [怎麼建立與編輯課程銷售頁 點我去看教學](/broken/pages/vMQLDl5jrC7nC6B5QlrH)

## <mark style="color:$primary;">課程訂價</mark>

### <mark style="color:blue;">課程訂價與購買備註</mark>

課程費用的設定與優惠都是在此處編輯

<figure><img src="../../.gitbook/assets/截圖 2025-08-12 下午3.57.59.png" alt=""><figcaption></figcaption></figure>



1. **原價** - 自訂課程費用<br>
2. **折扣價** - 自訂課程折扣價，如未設定則顯示原價<br>
3. **折扣價起始日** - 設定折扣價開始的日期與時間<br>
4. **折扣價到期日** - 設定折扣價結束的日期與時間<br>
5. **此課程是否為免費課程** - 如是免費課程，這裡要點擊按鈕開啟為藍色\
   \
   ❯❯ [如何建立免費課程](/broken/pages/fFDM4ce3uwA2srdX72XE)<br>
6. **購買備註** - 這裡可放一些提醒或要注意的文字\
   例如課程購買完成後記得去信箱收信，會有一封課前通知信要記得看完喔！\
   或是開學折扣季，ＯＯ課程30天限期快閃優惠別錯過！

### <mark style="color:blue;">觀看期限</mark>

**開課時間** - 可自定義開課日期與時間

**預計時長** - 此課程所有單元影片時長

**觀看期限** - 學員購買此課程後可有多久的觀看期限 固定天數單位有日、月、年 指定時間可選擇要在哪一天的幾點到期

**課程免費試看影片** - 會出現在課程頁面 簡介 內容的下方







### <mark style="color:blue;">ＱＡ設定</mark>

點 **新增** 按鈕會出現 輸入問題 的欄位

如需要調整QA項目的順序，點住前方的 ⠿ 圖標即可拖拉排序

<figure><img src="../../.gitbook/assets/截圖 2024-10-16 下午3.55.53.png" alt=""><figcaption></figcaption></figure>

### <mark style="color:blue;">其他設定</mark>

按鈕為灰色代表關閉，藍色則為啟用

<div align="left"><figure><img src="../../.gitbook/assets/截圖 2024-10-16 下午4.00.01.png" alt="" width="563"><figcaption></figcaption></figure></div>

1. **課程標籤** - 可選擇該課程是否為 熱門/精選 類別的課程<br>
2. **評價星星** - 可自定義顯示的評價星數\
   **灌水課程評價數量** - 預設為0，輸入的數字會顯示在星星後方

<div align="left"><figure><img src="../../.gitbook/assets/截圖 2024-10-16 下午4.07.12.png" alt="" width="563"><figcaption></figcaption></figure></div>

3. **顯示課程評價分頁** - 要不要將 評價的分頁 顯示在課程頁面中\
   **顯示用戶課程評價** - 要不要將 用戶評價 顯示在課程頁面中\
   **開放已購買用戶評價課程** - 是否限制只有購買該課程的學員才可留評價<br>
4. **課程學員人數** - 要不要顯示 學員人數 在課程頁面中\
   **灌水學員人數** - 如想要讓課程看起來很厲害可在這裡自定義數字，如果有新學員報名會自動加總灌水的數字反應在前台的 課程學員人數 上<br>
5. **課程留言** - 要不要開放課程頁面上的留言板\
   目前尚未提供留言編輯、隱藏留言的功能，如果有需要可以跟我們許願



到這裡新課程必設定的內容就完成了，

最後記得一定要點擊右上角的 「**儲存**」，才會呈現在前台喔！
